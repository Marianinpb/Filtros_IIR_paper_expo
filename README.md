# Expo Filtros IIR - Prueba de Concepto del SDD Harness

Este repositorio contiene los archivos y artefactos de una exposición académica sobre el paper *"Optimal Design of Digital IIR Filters Using Robust Group Search Optimization"*.

**Nota Importante:** Este proyecto sirve como la **primera prueba conceptual** del uso del `SDD Harness` como un generador estructurado de presentaciones académicas. Todas las fases de especificación, planificación e implementación se realizaron a través de este marco de trabajo iterativo.

## Estructura del Repositorio
- `presentacion/`: Contiene el código fuente LaTeX y el PDF compilado.
- `guion_exposicion.md`: Guion narrativo detallado para la exposición oral.
- `resumen_paper.md`: Análisis y extracción de los conceptos clave del artículo original.
- `.antigravity/`, `specs/`, `plans/`, `tasks/`, `memory/`: Artefactos y registros del proceso de desarrollo bajo el SDD Harness.
